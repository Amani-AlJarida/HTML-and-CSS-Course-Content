# HTML-and-CSS-Course-Content

Welcome to my web development repository! This project showcases my journey in front-end development.

## Projects Included

### 1. Bootstrap Cinema Website
A movie theater landing page featuring:
* **Bootstrap 5** integration.
* Interactive **Card Decks** for movie listings.
* **Pill Badges** used to display dynamic showtimes.
* Internal page linking (bookmarks) for easy navigation.

### 2. Personal Student Portfolio
A "Cyber-Gradient" themed portfolio that demonstrates core CSS and HTML skills:
* **Video Overlays:** A background video that autoplays on a loop.
* **Advanced CSS Filters:** Usage of `sepia` and `hue-rotate` on profile images.
* **Fixed Navigation:** A menu that stays at the top of the screen with hover effects.
* **Responsive Layouts:** A custom two-column layout and styled data tables.
* **Interactive Elements:** Image overlay fade effects and text hover animations.

## Technologies Used
* HTML5
* CSS3 (Flexbox, Transitions, Filters)
* Bootstrap 5
